# Trader Behavior vs Market Sentiment Analysis

## Objective
To analyze how Bitcoin market sentiment (Fear vs Greed) impacts trader performance and behavior.

## Methodology
- Data cleaning and preprocessing
- Date alignment and merging
- Feature engineering (PnL, win rate, trade frequency)
- Sentiment comparison
- Trader segmentation

## Key Insights
1. Average profitability remains similar across Fear and Greed regimes.
2. High-risk traders drive most of the returns.
3. Infrequent traders outperform frequent traders, suggesting overtrading reduces performance.
4. Fear periods show higher volatility.

## Strategy Recommendations
1. Avoid overtrading: Infrequent traders show superior performance.
2. Apply stricter risk controls during Fear periods due to increased volatility.

## Tools Used
Python, Pandas, Seaborn, Matplotlib
